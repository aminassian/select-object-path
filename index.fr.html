Présentation

sélectionner et extraire les chemins d'un objet json

Installation

Ce module est disponible sur npm : select-object-path 

$ npm install select-object-path

Utilisation

Le paramètre de la fonction sop.select doit respecter le schéma json select.in.schema.json.

var sop = require('select-object-path');
var result = sop.select({
    "select" : "-*,ccc", // exclude all except ccc
    "object": {
        "aaa": {
            "bbb" : {
                "ccc" : null
            }
        },
        "ccc": 10
    }
});

sop.select retourne un résultat qui respecte le schéma json select.out.schema.json.

{
    "path": [
        "/aaa/bbb/ccc",
        "/ccc"
    ],
    "licence": "GPL-3.0",
    "donate": [
        "https://pledgie.com/campaigns/28037",
        "https://gratipay.com/aminassian"
    ]
}

Test

Pour exécuter les tests : téléchargez le code source, installez les dépendances puis lancez la commande `npm test`. 

$ git clone https://github.com/aminassian/select-object-path.git
$ cd select-object-path
$ npm install
$ npm test
